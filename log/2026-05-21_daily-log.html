03The Knowledge

3.1 光速不變的 Canvas 呈現

用戶問了一個物理問題：「光速不變又怎呈現在 Canvas 上面？」 這不是純粹視覺問題，是概念問題。

技術實現

光子（Photon）類別的運動邏輯：

  • 每幀計算黑洞引力加速度，改變速度向量方向
  • 立即強制歸一化：vx² + vy² = C²（恆定）
  • 結果：路徑彎曲，但速率不變
  • 視覺上：軌跡呈拋物線或雙曲線，頭部光點恆定亮度

這是「光速不變」的程式化隱喻——不是精確物理模擬， 而是讓觀者「感覺到」某種不可違抗的恆常性。

3.2 重力透鏡的簡化模型

真實的愛因斯坦偏轉角 α = 4GM/(c²b)，需要廣義相對論計算。 Canvas 2D 上用了極簡化版本：

對每顆背景星星，計算其到黑洞的距離 r， 視覺偏移量 = k × M / r，方向為遠離黑洞。 這產生愛因斯坦環的視覺印象——星星被「推開」形成環狀結構。

3.3 Deploy 腳本的 category 擴充

腳本原本只支援 log/guide/artwork，加入 knowledge 需要改四處： docstring、驗證 tuple、錯誤訊息、update_index() 的 categories dict。 關鍵洞察：腳本和 repo 裡的 scripts/ 是兩個獨立副本， 本地執行的是 ~/scripts/，repo 裡的是備份。
